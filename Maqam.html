Al-Aseel Maqam

A professional Android application for musicians who need precise quarter-tone tuning, advanced MIDI routing, and live transpose control for arranger and workstation keyboards.

Download on Google Play

Features

• Professional ¼-tone tuning with cent value control
• Adjustable cent precision for detailed maqam performance
• Badel Router for MIDI routing between devices
• Real-time transpose with instant reset
• Multi-keyboard support (control two keyboards simultaneously)
• Five preset save buttons for instant recall
• Optimized for live stage performance

Hardware Requirements

To connect two keyboards at the same time you need:
• USB Hub or OTG Hub
• Two USB printer cables (USB-A to USB-B)

العربية

تطبيق الأصيل مقام هو أداة احترافية للتحكم بالربع تون بدقة عالية، مع إمكانية التحكم بقيمة السنت (Cent)، وتوجيه إشارات MIDI عبر راوتر البدل، وإدارة الترانسبوز أثناء العزف المباشر.

المميزات:
• تحكم كامل بقيمة السنت بدقة عالية
• دعم الربع تون للمقامات الشرقية
• راوتر البدل لتوجيه MIDI بين الأجهزة
• ترانسبوز فوري مع زر رجوع للصفر
• دعم تشغيل جهازين في نفس الوقت
